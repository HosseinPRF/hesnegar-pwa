# حس‌نگار (PWA آفلاین)

دو مسیر در یک اپ:

- **ثبت سریع (۶۰–۹۰ ثانیه)**: نام‌گذاری احساس، شدت، بدن، تریگر، جملهٔ ذهن، انتخاب یک تکنیک، اقدام کوچک.
- **تحلیل عمیق (۷–۱۲ دقیقه)**: واقعیت/داستان، احساس‌های زیرین، نیازهای تئوری انتخاب، گفتگوی درونی (TA)، ریشه/آشنایی، انتخاب تکنیک، پلان، ارزیابی.

همه داده‌ها فقط داخل **localStorage** دستگاه شما ذخیره می‌شود.

## اجرا

### گزینه ۱: باز کردن مستقیم
فایل `index.html` را باز کنید.

### گزینه ۲: نصب PWA
بهترین تجربه وقتی است که با یک سرور محلی اجرا کنید تا Service Worker فعال شود:

```bash
# با npm
npx serve .
```

سپس در مرورگر (Chrome/Edge) گزینه **Install** را بزنید.

## خروجی/ورودی
- از دکمه **خروجی JSON** برای دانلود همه رکوردها استفاده کنید.
- از **ورود JSON** برای وارد کردن فایل خروجی قبلی.
